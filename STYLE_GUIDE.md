# Brand Style Guide

## Colors

### Primary Colors
- **Deep Navy Blue**: `#1A365D` - Primary brand color
- **Luxe Gold**: `#D4AF37` - Accent color

### Secondary Colors
- **White**: `#FFFFFF` - Text on dark backgrounds
- **Light Gray**: `#F7FAFC` - Light backgrounds
- **Medium Gray**: `#A0AEC0` - Secondary text
- **Dark Gray**: `#2D3748` - Dark text on light backgrounds

### Status Colors
- **Success**: `#38A169` - Success states
- **Warning**: `#D69E2E` - Warning states
- **Error**: `#E53E3E` - Error states
- **Info**: `#3182CE` - Information states

## Typography

### Fonts
- **Headers (H1-H6)**: Rubik, Arial, sans-serif
- **Body Text**: Quicksand (Google Fonts), sans-serif
- **Code**: Monaco, Consolas, monospace

### Font Sizes
- **H1**: 2.5rem (40px)
- **H2**: 2rem (32px)
- **H3**: 1.75rem (28px)
- **H4**: 1.5rem (24px)
- **H5**: 1.25rem (20px)
- **H6**: 1.125rem (18px)
- **Body Large**: 1.125rem (18px)
- **Body**: 1rem (16px)
- **Body Small**: 0.875rem (14px)
- **Caption**: 0.75rem (12px)

### Font Weights
- **Light**: 300
- **Regular**: 400
- **Medium**: 500
- **Semibold**: 600
- **Bold**: 700

## Spacing & Layout

### Spacing Scale (rem)
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)
- **2xl**: 3rem (48px)
- **3xl**: 4rem (64px)
- **4xl**: 6rem (96px)

### Padding
- **Component Padding**: 1rem (16px)
- **Section Padding**: 2rem (32px)
- **Page Padding**: 1.5rem (24px) mobile, 3rem (48px) desktop

### Margins
- **Element Margin**: 0.5rem (8px)
- **Section Margin**: 2rem (32px)
- **Component Margin**: 1rem (16px)

## Border Radius
- **Small**: 0.25rem (4px)
- **Medium**: 0.5rem (8px)
- **Large**: 0.75rem (12px)
- **XL**: 1rem (16px)
- **Full**: 9999px (circular)

## Visual Effects
- **Shadows**: Not used - maintain clean, flat design
- **Gradients**: Not used - use solid colors only
- **Effects**: Prefer subtle borders and color transitions

## Breakpoints
- **Mobile**: 320px - 767px
- **Tablet**: 768px - 1023px
- **Desktop**: 1024px - 1439px
- **Large Desktop**: 1440px+

## Component Guidelines

### Buttons
- **Height**: 2.5rem (40px) default, 2rem (32px) small, 3rem (48px) large
- **Padding**: 0.75rem 1.5rem (12px 24px)
- **Border Radius**: 0.5rem (8px)

### Input Fields
- **Height**: 2.5rem (40px)
- **Padding**: 0.75rem (12px)
- **Border**: 1px solid
- **Border Radius**: 0.5rem (8px)

### Cards
- **Padding**: 1.5rem (24px)
- **Border Radius**: 0.75rem (12px)
- **Border**: 1px solid light gray instead of shadows

## Animation
- **Duration**: 200ms (fast), 300ms (normal), 500ms (slow)
- **Easing**: ease-in-out (default), ease-out (entrances), ease-in (exits)

## Accessibility
- **Minimum contrast ratio**: 4.5:1 for normal text, 3:1 for large text
- **Focus indicators**: 2px solid accent color with 2px offset
- **Touch targets**: Minimum 44px x 44px
